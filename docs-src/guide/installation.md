# インストール

## 必要環境

- Unity 6000.0.72f1 (Unity 6 LTS) 以降
- Android プラグイン / iOS プラグイン (書き出し先に応じて)
- 日本語言語パック (推奨)

::: tip
インストール手順が分からない場合は [UnityHub のインストール解説記事](https://creive.me/archives/13376/) も参考になります。9 GB 以上の空き容量が必要です。
:::

## 手順

1. [UnityHub](https://unity3d.com/jp/get-unity/download) を入手し、必要なモジュールと一緒に Unity をインストールします。
2. [GameCanvas の最新リリース](https://github.com/sfc-sdp/GameCanvas-Unity/releases/latest) からソースコードを入手・解凍します。
3. UnityHub の「リストに追加」から解凍したフォルダを選択すると、プロジェクトが登録されます。
4. プロジェクトを開き、エディターが立ち上がったらプレビューボタン (▶️) を押します。サンプルが実行されれば導入完了です。
5. `Assets/Game.cs` を編集して、自分だけのオリジナルアプリを作りましょう。
