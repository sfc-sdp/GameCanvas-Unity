Priority

アクターの処理優先度

Declaration
int Priority { get; set; }
Property Value
Type Description
Int32
Remarks
  • 初期値は0（無指定）
  • アクターは、シーン内でこの値が小さい順に処理が回ってきます
  • 同じ値だった場合は、シーンに登録した順序で実行されます
  • 負の値を設定した場合は、そのシーンの処理よりも前に実行されます
